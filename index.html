Visualizing IT security incidents

This is my final project of the minor Programming at the University of Amsterdam. I personally think there is not enough open source data available about IT security incidents. We hear alot about cyber security and hacking in the news, but what about it world wide? With the VERIS community database I tend to offer some overview into the overload of information about this topic. The dataset I used for this visualization is last updated on 28-12-2017.

About the data source

These visualizations are based on the Vocabulary for Event Recording and Incident Sharing (VERIS) database. This database is mantained by volunteers, that add IT security incidents to the open source database in a uniform format. Note that this is one of the few public available data resources on this topic. Data represented is fully based on voluntairy work. For more information about the data source, please check their website.
